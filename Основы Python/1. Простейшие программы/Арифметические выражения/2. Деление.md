==" / "== - классическое деление:
```py
a = 9; b = 6
x = 3 / 4 # = 0.75
x = -a / b # = -1.5
```

==" // "== - целочисленное деление (округление "вниз"!):
```py
a = 9; b = 6
x = a // b # = 1
x = -3 // 4 # = -1
```
==" % "== - остаток от деления
```py
d = 85
b = d // 10 # = 8
a = d % 10 # = 5
d = a % b # 5
d = b % a # 3
```
Для отрицательных чисел:
```py
b = -7 // 2 # -4
d = -7 % 2 # 1
```
