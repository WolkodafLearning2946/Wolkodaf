> [!warning]+ Целая и дробная части числа разделяются точкой!

> [!example]+ Код
> ```python
> x = 123.456
> print(x)
> print("{:10.2f}".format(x)) #10 - всего знаков. 2f - в дробной части знаков
> ```
> > [!abstract]+ Вывод:
> > ```py
> > 123.456
> > ....123.46 # число в дробной части округляется
> > ```

> [!example]+ Код
> ```python
> x = 123.456
> print("{:10.2g}".format(x))
> ```
> > [!abstract]+ Вывод:
> > ```py
> > ...1.2e+02
> > ```
