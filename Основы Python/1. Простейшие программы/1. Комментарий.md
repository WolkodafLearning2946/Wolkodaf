Закомментировать одну строку кода можно с помощью ==#== :

> [!example]+ Пример
> ```py
>	#Этот текст не читается
>

Несколько же строк комментируют с помощью трёх апострофов =='''== :

> [!example]+ Пример
> ```py
> 	'''
> 	Код не читается
> 	Код не читается!
> 	Код не читается!!
> 	Код не читается!!!
> 	'''
